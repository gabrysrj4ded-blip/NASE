# NASE Project Rules

## Naming

Folders:
lowercase

Components:
PascalCase

Hooks:
useSomething

Constants:
lowercase

---

## Code Rules

- No hardcoded colors
- No magic numbers
- One responsibility per component
- Reusable components only
- Responsive first
- Clean code
- Maximum component size ≈ 200 lines

---

## Architecture

Business logic must not be placed inside app/

Use constants whenever possible.

All texts must support localization.
